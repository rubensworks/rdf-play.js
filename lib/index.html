RDF Play

This is a Web-based tool for performing simple client-side RDF operations. Currently, dereferencing RDF by URL with different result formats is supported (defaults to TriG).

All major RDF 1.2 serializations are supported, such as Turtle, JSON-LD, RDFa in HTML, script data blocks in HTML, ...
Internally, this tool makes use of RDF parsers from the Comunica framework, which enable streaming processing of RDF.

Want to run this from the command-line? Have a look at rdf-dereference, rdf-parse, and rdf-serialize.
